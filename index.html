Minecraft Web Client

Overview & Instructions

Welcome to the Minecraft Web Client project. This interface allows you to test UI components, interact with the backend, and verify that all systems are functioning correctly.

Authentication with Microsoft, Xbox Live, and XSTS is fully implemented on the backend. However, Minecraft-specific authentication requires Mojang API approval.

Microsoft Authentication

Begin the authentication chain by signing in with your Microsoft account.

Xbox Live Authentication

Exchange your Microsoft token for an Xbox Live token.

XSTS Authorization

Authorize your Xbox Live identity with XSTS.

Minecraft Authentication

This step normally exchanges your XSTS token for a Minecraft access token.

Notice:
Minecraft authentication is not yet available. Mojang requires manual approval for third‑party applications accessing Java Edition game services. Integration will be enabled once API access is granted.
